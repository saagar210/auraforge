# AuraForge Distribution

This folder contains everything needed to install and use AuraForge.

## Contents

| File | Description |
|------|-------------|
| **AuraForge_0.1.0_aarch64.dmg** | macOS installer (Apple Silicon) |
| **INSTALL.md** | Step-by-step installation guide |
| **QUICK_REFERENCE.md** | Keyboard shortcuts and workflow cheat sheet |

## Quick Start

1. Install [Ollama](https://ollama.com/download) and open it
2. Double-click the **.dmg** file and drag AuraForge to Applications
3. Open AuraForge and follow the setup wizard

See **INSTALL.md** for detailed instructions.

## Requirements

- macOS (Apple Silicon or Intel)
- ~20 GB free disk space for the AI model
- Internet connection for initial model download

## Version

AuraForge 0.1.0
